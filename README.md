<h1>Chess AI</h1>

<h4>Requirements</h4>

 - display - 1920x1080

 - os - Linux

<h4>Install</h4>

- install chome extension from extensions/chrome

- pipenv install

<h4>Run</h4>

- pipenv run main

- start chess game and click to extension icon 

<h4>Troubleshooting:</h4>

 - sometimes freezes -> need reload browser tab and click to extension icon
